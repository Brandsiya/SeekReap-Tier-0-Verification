# Contributing to SeekReap Tier 0

Thank you for your interest in SeekReap.

## ⚠️ Important: This Repository Represents Tier 0
Tier 0 is a **frozen, deterministic foundation**. It is intentionally limited and does not evolve feature-wise.

## What Contributions Are Accepted

### ✅ Allowed
- Documentation corrections or clarifications
- Typographical fixes
- Improvements to examples that **do not change behavior**
- Verification of deterministic outputs

### ❌ Not Accepted
- New features or extensions
- Verification engines or logic
- Networking, APIs, SDKs, or telemetry
- UI, dashboards, or web services
- Commercial integrations
- Any PR that violates the Tier 0 boundaries

**Any Pull Request that violates these boundaries will be closed without review.**

## How to Contribute
1. Fork this repository
2. Create a branch (e.g., `docs/fix-typo`, `examples/clarify`)
3. Make your change
4. Submit a Pull Request with a clear description

## Feature Requests
**Tier 0 does not accept feature requests.**  
Requests related to verification, APIs, SDKs, or integrations belong to **Tier 1+** and will be redirected.

---

By contributing, you acknowledge that **Tier 0 is frozen by design** and exists to establish boundaries, not to evolve.
