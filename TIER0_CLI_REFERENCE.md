# CLI Reference

## Commands

### verify
```bash
seekreap-verify verify <file>
```
Verifies a policy file for syntax and basic validation.

### check
```bash
seekreap-verify check <file>
```
Checks file syntax without full validation.

### version
```bash
seekreap-verify version
```
Shows version and boundary information.

### help
```bash
seekreap-verify help
```
Shows help message.

## Options
- `--strict`: Enable strict validation mode
- `--format <type>`: Output format (text|json)
