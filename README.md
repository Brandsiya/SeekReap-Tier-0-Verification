# SeekReap Tier 0 Verification

Open Source Tier 0 Boundary Verification System

## Quick Start

\`\`\`bash
# Install dependencies
npm install

# Test installation
node cli.js version

# Verify a file
node cli.js verify examples/test.json
\`\`\`

## Commands

- \`node cli.js version\` - Show version
- \`node cli.js verify <file>\` - Verify a file
- \`node cli.js --help\` - Show all commands
