# SeekReap Tier 0

## Overview
Tier 0 is a boundary-controlled CLI tool for deterministic policy verification.

## Quick Start
1. Clone this repository
2. Build from source: `npm install && npm run build`
3. Verify a policy: `./dist/cli verify examples/basic-policy.json`
